# SBM Safety Academy

Portal de microcapacitaciones EHS para GitHub Pages con registro digital en Supabase.

## Archivos principales
- `index.html`: página principal.
- `styles.css`: diseño.
- `app.js`: lógica y conexión a Supabase.
- `videos/`: videos MP4.
- `docs/`: formato RH-F-05.
- `supabase_setup.sql`: tabla y políticas de Supabase.

## Antes de publicar
1. En Supabase, abrir SQL Editor.
2. Copiar y ejecutar el contenido de `supabase_setup.sql`.
3. Subir todo el contenido de esta carpeta al repositorio GitHub.
4. Activar GitHub Pages desde `Settings > Pages > Deploy from branch > main > /root`.

## Nota
El registro digital es apoyo de seguimiento. La evidencia oficial sigue siendo el formato RH-F-05 firmado.
